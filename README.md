# DungeonVault4e

This project was bootstrapped with [Vite](https://vitejs.dev/guide/).

## Dependencies

- Node v20.16.0 (npm v10.8.1)

## Running Locally

Run the app in the development mode:

1. `npm install`
2. `npm start`

The application should now be accessible at [http://localhost:5173](http://localhost:5173).

## Running in Docker

Build and run the Docker image:

1. `docker build -t DungeonVault4e .`
2. `docker run --name DungeonVault4e -d -p 5173:80 DungeonVault4e`

The application should now be accessible at [http://localhost:5173](http://localhost:5173).

## Running in Kubernetes

Run the Jenkins Job for this project: [Jenkins Job]().
The application should now be accessible at [SITE_ADDRESS]().
